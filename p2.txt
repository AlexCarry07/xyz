import java.util.*;

class Astar {
    public static void main(String[] args) {

        int graph[][] = {
            {0, 2, 4, 0},
            {2, 0, 1, 7},
            {4, 1, 0, 3},
            {0, 7, 3, 0}
        };

        // Heuristic values
        int h[] = {7, 6, 2, 0};

        int n = 4;
        int start = 0;
        int goal = 3;

        boolean visited[] = new boolean[n];
        int current = start;
        int cost = 0;

        System.out.print("Path: " + current);

        while (current != goal) {

            visited[current] = true;

            int min = 999;
            int next = -1;

            System.out.println("\n\nChecking neighbors of " + current + ":");

            for (int i = 0; i < n; i++) {

                // Check connected unvisited node
                if (graph[current][i] != 0 && !visited[i]) {

                    // f(n) = edge cost + heuristic
                    int f = graph[current][i] + h[i];

                    System.out.println("Node " + i + " -> f = " + f);

                    // Select minimum f(n)
                    if (f < min) {
                        min = f;
                        next = i;
                    }
                }
            }

            // Add path cost
            cost += graph[current][next];

            current = next;

            System.out.println("\nSelected Node: " + current);
        }

        System.out.println("\n\nDestination Reached");
        System.out.println("Final Cost = " + cost);
    }
}

The graph stores all nodes and path costs between nodes.
Heuristic values estimate how close each node is to the goal node.
The algorithm starts from the source node.
A visited array is used to avoid visiting the same node again.
The algorithm checks all neighboring nodes connected to the current node.
For every neighbor, it calculates:
f(n)=g(n)+h(n)

Where:

	g(n) = path cost
	h(n) = heuristic value
	f(n) = total estimated cost
The node with the smallest f(n) value is selected.
The algorithm moves to that selected node.
Total path cost is updated.
Steps continue until the goal node is reached.
Finally, the program prints:
Selected path
Destination reached
Total cost
