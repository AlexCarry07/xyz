import java.util.*;
class GraphTraversal{
    static int graph[][],visited[],n;

    // DFS using recursion
    static void DFS(int node){
        visited[node]=1; // mark visited
        System.out.print(node+" ");

        for(int i=0;i<n;i++){
            // visit connected unvisited node
            if(graph[node][i]==1 && visited[i]==0){
                DFS(i);
            }
        }
    }

    // BFS using queue
    static void BFS(int start){
        Queue<Integer> q=new LinkedList<>();
        int visit[]=new int[n];

        q.add(start); // add start node
        visit[start]=1;

        while(!q.isEmpty()){
            int node=q.poll(); // remove node
            System.out.print(node+" ");

            for(int i=0;i<n;i++){
                // visit connected unvisited node
                if(graph[node][i]==1 && visit[i]==0){
                    q.add(i);
                    visit[i]=1;
                }
            }
        }
    }

    public static void main(String args[]){
        Scanner sc=new Scanner(System.in);

        System.out.println("Enter number of nodes:");
        n=sc.nextInt();

        graph=new int[n][n];
        visited=new int[n];

        // input adjacency matrix
        System.out.println("Enter adjacency matrix:");
        for(int i=0;i<n;i++){
            for(int j=0;j<n;j++){
                graph[i][j]=sc.nextInt();
            }
        }

        System.out.println("DFS:");
        DFS(0);

        System.out.println("\nBFS:");
        BFS(0);

        sc.close();
    }
}

This Java program performs graph traversal using two algorithms:
DFS (Depth First Search)
BFS (Breadth First Search)

The graph is represented using an adjacency matrix.

First, the program imports the Java utility package which contains Scanner, Queue, and LinkedList classes.

The class name is GraphTraversal.
Three static variables are declared:
graph[][] → stores the adjacency matrix
visited[] → keeps track of visited nodes in DFS
n → stores number of nodes

DFS FUNCTION:
The DFS function is recursive.
It takes one node as input.
First, the node is marked as visited.
Then the node is printed.
After that, a loop checks all nodes connected to the current node.
If a node is connected and not visited, DFS is called again for that node.
DFS works by going deep into one path before coming back.

BFS FUNCTION:
The BFS function uses a queue.
A queue and a local visited array are created.
The starting node is added into the queue and marked visited.
A while loop runs until the queue becomes empty.
Inside the loop:
the front node is removed using poll()
the node is printed
all adjacent unvisited nodes are added into the queue
BFS visits nodes level by level.

MAIN FUNCTION:
The program starts execution from main().
First, the user enters the number of nodes.
Then memory is allocated for:
adjacency matrix
visited array
Next, the adjacency matrix is entered by the user.
After taking input:
DFS starts from node 0
BFS starts from node 0
Finally, Scanner is closed.

ADJACENCY MATRIX:
If graph[i][j] = 1:
there is an edge between node i and node j
If graph[i][j] = 0:
there is no edge
