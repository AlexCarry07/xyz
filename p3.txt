import java.util.*;

class SelectionSort {
    public static void main(String args[]) {

        Scanner sc = new Scanner(System.in);

        // Input number of elements
        System.out.println("Enter no of elem");
        int n = sc.nextInt();

        // Input array elements
        System.out.println("Enter elem");
        int a[] = new int[n];

        for (int i = 0; i < n; i++) {
            a[i] = sc.nextInt();
        }

        // Selection Sort
        for (int i = 0; i < n - 1; i++) {

            int min = i; // assume current index is minimum

            // Find smallest element
            for (int j = i + 1; j < n; j++) {
                if (a[j] < a[min]) {
                    min = j;
                }
            }

            // Swap elements
            int temp = a[i];
            a[i] = a[min];
            a[min] = temp;
        }
	System.out.print("selection sort : ");

        // Print sorted array
        for (int x : a) {
            System.out.print(x + " ");
        }
    }
}























































































































































The program first asks the user to enter the number of elements in the array.
Then it takes all the array elements as input from the user.
After taking input, the Selection Sort algorithm starts.
How Selection Sort Works
The algorithm assumes the first element is the smallest.
It then compares this element with all remaining elements in the array.
If a smaller element is found, its position is stored.
After checking all elements, the smallest element is swapped with the current element.
This process repeats for every position in the array until the array becomes sorted.
Example

If the array is:
64 25 12 22 11

In the first pass, 11 is the smallest, so it is placed at the first position.
In the second pass, 12 is the smallest among remaining elements, so it is placed at the second position.
The process continues until all elements are arranged in ascending order.
Final Output

The program prints the sorted array in ascending order.

































































































































































































































import java.util.*;

class Prims {

    public static void main(String args[]) {

        // Graph using adjacency matrix
        int graph[][] = {
            {0, 2, 0, 6, 0},
            {2, 0, 3, 8, 5},
            {0, 3, 0, 0, 7},
            {6, 8, 0, 0, 9},
            {0, 5, 7, 9, 0}
        };

        int n = 5;

        // Track visited vertices
        boolean visited[] = new boolean[n];

        // Start from vertex 0
        visited[0] = true;

        System.out.println("Edges in MST:");

        // Find n-1 edges
        for (int count = 0; count < n - 1; count++) {

            int min = 999;
            int x = 0, y = 0;

            // Find minimum edge
            for (int i = 0; i < n; i++) {

                if (visited[i]) {

                    for (int j = 0; j < n; j++) {

                        // Check unvisited connected vertex
                        if (!visited[j] && graph[i][j] != 0) {

                            // Select smaller edge
                            if (graph[i][j] < min) {
                                min = graph[i][j];
                                x = i;
                                y = j;
                            }
                        }
                    }
                }
            }

            // Print selected edge
            System.out.println(x + " - " + y + " : " + graph[x][y]);

            // Mark vertex as visited
            visited[y] = true;
        }
    }
}












































































































































































































































































Prim’s Algorithm – Simple Explanation
The program is used to find the Minimum Spanning Tree (MST) of a graph.
The graph is represented using an adjacency matrix.
The value in the matrix represents the weight of the edge between two vertices.
A visited array is used to store visited vertices.
The algorithm starts from vertex 0.
The starting vertex is marked as visited.
The program checks all edges connected to visited vertices.
It selects the edge with minimum weight.
The selected edge is added to the Minimum Spanning Tree.
The new vertex connected by that edge is marked as visited.
The process repeats until all vertices are visited.
Finally, the program prints all edges of the MST.
The MST connects all vertices with minimum total cost.
Prim’s Algorithm is called a greedy algorithm because it chooses the minimum edge at every step.
